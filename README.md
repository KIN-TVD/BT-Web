# BT-Web

LT-WEB /                               Tập tin,Mô tả chi tiết
📁 CSS/                                 Thư mục chứa toàn bộ các file định dạng giao diện cho website.
├── 📁 themify-icons/                   Bộ icon font hệ thống dùng để hiển thị các biểu tượng đồ họa.
│   ├── 📁 demo-files/                  Các file ví dụ mẫu để xem cách sử dụng icon.
│   ├── 📁 fonts/                       Chứa các file font thực tế (eot, ttf, woff)."
│   ├── 📁 SVG/                         Các định dạng ảnh vector của icon.
│   ├── themify-icons.css               File CSS chính để nhúng bộ icon vào trang web.
│   └── Themify IconFonts...json        File cấu hình chứa dữ liệu mã hóa của bộ icon.
├── style.css                           "File định nghĩa phong cách (layout, màu sắc, font chữ) chính cho dự án."
index.html                              Trang chủ và là tệp tin khởi chạy chính của trang web.
script.js,                              File JavaScript xử lý các hiệu ứng và tính năng tương tác cho website.
README.md,                              File tài liệu mô tả dự án (hiển thị trên GitHub).
