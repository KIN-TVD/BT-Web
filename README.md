#  LT-WEB

## 📂 Cấu trúc thư mục (Project Structure)

Dưới đây là sơ đồ chi tiết các thành phần trong mã nguồn:

| Thành phần | Loại | Mô tả |
| :--- | :---: | :--- |
| 📁 **CSS/** | Folder | Chứa toàn bộ tệp tin định dạng giao diện |
| ├── 📂 **themify-icons/** | Lib | Bộ sưu tập Icon Font chuyên nghiệp |
| │   ├── 🔠 `fonts/` | Assets | Chứa các định dạng font (eot, ttf, woff) |
| │   └── 🎨 `SVG/` | Graphics | Định dạng vector cho các biểu tượng |
| ├── 📄 `style.css` | File | Tệp tin CSS chính quản lý Layout & Style |
| ├── 📄 `themify-icons.css` | Style | Thư viện liên kết các Icon vào dự án |
| 📄 **index.html** | HTML | Trang chủ và là tệp tin khởi chạy chính |
| 📄 **script.js** | JS | Xử lý các hiệu ứng và tính năng tương tác |
| 📝 **README.md** | Doc | Tài liệu hướng dẫn và thông tin dự án |

---

## 🛠️ Công nghệ sử dụng

* **Frontend:** HTML5, CSS3 (Flexbox & Grid).
* **Icons:** Themify Icons Library.
* **Scripting:** Vanilla JavaScript.
